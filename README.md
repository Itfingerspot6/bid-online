# 🔨 BidOnline - Modern Auction Platform

<p align="center">
    <img src="https://raw.githubusercontent.com/laravel/art/master/logo-lockup/5%20SVG/2%20CMYK/1%20Full%20Color/laravel-logolockup-cmyk-red.svg" width="300" alt="Laravel Logo">
</p>

**BidOnline** adalah platform lelang online premium yang dirancang untuk kecepatan, keamanan, dan interaktivitas real-time. Dibangun dengan teknologi terbaru dari ekosistem Laravel.

## 🚀 Fitur Unggulan
- ⚡ **Real-time Price Engine**: Harga update seketika tanpa refresh.
- 🤖 **Auto-Bid (Proxy Bidding)**: Biarkan sistem menawar untukmu.
- 💳 **Seamless Payment**: Integrasi deposit saldo via Midtrans.
- 🎨 **Premium UI**: Tampilan dark mode yang elegan dengan Tailwind & Alpine.js.
- 🏆 **Win & Refund System**: Logika pemenang dan pengembalian dana otomatis yang aman.

## 🛠 Instalasi Cepat
1. Clone repository
2. `composer install` & `npm install`
3. Salin `.env.example` ke `.env` dan konfigurasi database.
4. `php artisan migrate --seed`
5. `php artisan key:generate`
6. Jalankan server: `php artisan serve`, `npm run dev`, dan `php artisan reverb:start`.
